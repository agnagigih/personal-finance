# Personal Finance API

Backend API for a personal finance application.

## Features
- JWT authentication
- Account, category, transaction management
- User-based data isolation
- Pagination
- Global exception handling

## Tech Stack
- ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT Authentication
- FluentValidation

## Architecture
- Controllers: HTTP layer
- Services: business logic
- DTOs: request/response models
- Middleware: cross-cutting concerns

This project is built for learning and backend practice.
